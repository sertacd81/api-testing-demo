\# API Testing Demo (Anonymized)



This is a small demo of a digital manufacturing API test flow.

It uses \*\*Postman\*\* for request design and \*\*Node.js (axios, dotenv, csv-parser)\*\* for automation.



\## Features

\- CSV-driven data input

\- Unique transaction/serial generation

\- JSON logging and basic error handling



\## How to run

```bash

npm install axios csv-parser dotenv

node runner.js
